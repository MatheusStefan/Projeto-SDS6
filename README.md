# Projeto-SDS6
## O projeto em questão, realizado pela DevSuperior, consiste em construir uma API REST de avaliações de nota de filmes.
### Tecnologias usadas:
- Java/SpringBoot
- React
- Heroku e Netlify
- GitHub para documentação
- Postgres
